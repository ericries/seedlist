## About

Gitar is a San Francisco-based developer infrastructure company building AI agents for code review and continuous integration workflows [^1]. The company emerged from stealth on April 15, 2026, founded by Ali-Reza Adl-Tabatabai, a veteran of Intel Labs, Google, and Uber, and Gautam Korlam [^1]. Gitar's platform provides "code validation" — a workflow agent that orchestrates reviews, tests, and diagnostics end to end, addressing the bottleneck shifting from writing code to validating it [^1].
