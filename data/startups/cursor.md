## About

Cursor is an AI-powered code editor built by Anysphere, founded in 2022 by Michael Truell, Sualeh Asif, Arvid Lunnemark, and Aman Sanger while students at MIT [^1]. The product is a fork of Visual Studio Code with integrated AI-assisted coding capabilities and is used by over 50% of the Fortune 500 [^2]. Bloomberg has described it as "the fastest growing startup ever" [^3].
