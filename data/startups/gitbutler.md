## About

GitButler is a developer infrastructure company building version control tools designed for modern AI-assisted coding workflows [^1]. Founded by GitHub co-founder Scott Chacon, the company develops a Git client that enables parallel and stacked branches, unlimited undo, easy commit editing, and seamless integration with AI agents like Claude and Cursor [^1][^2]. The platform is designed for trunk-based development workflows and aims to build what comes after Git [^1].
