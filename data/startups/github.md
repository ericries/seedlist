## About

GitHub is a web-based platform for software development and version control using Git, providing hosting for code repositories, collaboration tools, and CI/CD services [^1]. The service was developed by Chris Wanstrath, P.J. Hyett, Tom Preston-Werner, and Scott Chacon, launching in April 2008 [^2]. GitHub bootstrapped profitably for its first four years before taking outside funding [^3]. Microsoft acquired GitHub in October 2018 for $7.5 billion in an all-stock deal [^2].

## What Investors Say

**Andreessen Horowitz:** The $100 million Series A in 2012 was the largest single check Andreessen Horowitz had ever written at the time. The investment netted over $1 billion for the fund when Microsoft acquired GitHub for $7.5 billion [^3][^6].
